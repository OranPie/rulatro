Mad Joker is a Common Additive Mult Joker that provides +10 Mult when the played hand contains a Two Pair. This does not include hands like Four of a Kind, but it does include hands like Full House

Synergies

[edit | edit source]

Decks

[edit | edit source]
  • Abandoned Deck ( Start run with no {{F|}}[[Card Ranks#Face cards|{{hl|orange|Face Cards}}]] in your deck Win a run with the {{D|Green}} on [[Stakes|any difficulty]] Abandoned Deck.png) and Mad World ( ) work well with this Joker thanks to their smaller deck, allowing you to find a Two Pair easier.
  • Erratic Deck ( All [[Card Ranks|{{hl|orange|Ranks}}]] and [[Suits|{{hl|orange|Suits}}]] in deck are randomized Win a run with any [[Decks|deck]] on the {{Stake|Orange}} difficulty or harder Erratic Deck.png) can work well with this Joker if it randomizes a lot of cards with the same rank.

Poker Hands

[edit | edit source]

Any Poker Hand that contains a Two Pair will benefit from Mad Joker. These are:

  • Two Pair
  • Full House
  • Flush House
  • Flush - only if the flush contains a Two Pair

Four of a Kind and similar hands do not trigger Mad Joker.

Jokers

[edit | edit source]
  • Spare Trousers (This Joker gains {{Mult|+2}} if played hand contains a [[Poker Hands|{{hl|orange|Two Pair}}]] <br> {{currently|{{mult|+0}}}} Uncommon Additive Mult Mixed Spare Trousers.png) gains Mult when playing a Two Pair.
  • Square Joker (This Joker gains {{Chips|+4}} if played hand has exactly {{hl|orange|4}} cards <br> {{Currently|{{Chips|0}}}} Common Chips Mixed Square_Joker.png) gains Chips any time a played hand consists of exactly 4 cards. Since a standard Two Pair is exactly four cards, the player can strengthen Square Joker while getting Mult from Mad Joker.
  • Sly Joker ({{Chips|+50}} if played hand contains a {{ph|Pair}} Common Chips Independent Sly Joker.png) and Clever Joker ({{Chips|+80}} if played hand contains a {{ph|Two Pair}} Common Chips Independent Clever Joker.png) give Chips on a Pair and Two Pair respectively.
  • Jolly Joker ({{Mult|+8}} if played hand contains a {{ph|Pair}} Common Additive Mult Independent Jolly Joker.png) gives Mult if the played hand contains a Pair.

Trivia

[edit | edit source]
  • This joker is part of a series which provide a Mult boost for a specific hand. The others are Jolly Joker ({{Mult|+8}} if played hand contains a {{ph|Pair}} Common Additive Mult Independent Jolly Joker.png), Zany Joker ({{Mult|+12}} if played hand contains a {{ph|Three of a Kind}} Common Additive Mult Independent Zany Joker.png), Crazy Joker ({{Mult|+12}} if played hand contains a {{ph|Straight}} Common Additive Mult Independent Crazy Joker.png) and Droll Joker ({{Mult|+10}} if played hand contains a {{ph|Flush}} Common Additive Mult Independent Droll Joker.png).
