Obelisk is a Rare scaling Multiplicative Mult Joker that provides an additional X0.2 Mult each time a hand that is not one of the player's most-played hands is played. This Joker starts at X1 Mult and will reset to X1 Mult if the most-played hand is played. The Obelisk resets before the hand is scored.

You can safely play any hand type until it is tied to your most-played hand type. Obelisk will only reset once you break the tie between those hands (i.e., you play one of those hands one more time).

Synergies

[edit | edit source]

Obelisk can become a very powerful Scaling XMult Joker, but it cannot be used in every build. It works best on builds that don't rely on playing specific hands to score, as if too many of any one hand are played, you can no longer play it without resetting the Obelisk.

The most effective strategy with Obelisk is to play Pair or Two Pair until around 15 have been played, then pivoting in the later antes to playing any other hand. It is not recommended to use Obelisk if the most played hand is High Card, as the player runs a higher risk of being forced to reset the Obelisk.

Decks

[edit | edit source]
  • Red Deck ( {{hl|red|+1}} discard every round Unlocked from start Red Deck.png)'s extra discards, Painted Deck ( {{hl|orange|+2}} hand size, <br> {{hl|red|-1}} Joker slot Win a run with any [[Decks|deck]] on the {{Stake|Green}} difficulty or harder Painted Deck.png)'s higher hand size, and Abandoned Deck ( Start run with no {{F|}}[[Card Ranks#Face cards|{{hl|orange|Face Cards}}]] in your deck Win a run with the {{D|Green}} on [[Stakes|any difficulty]] Abandoned Deck.png)'s smaller deck size all allow the player to find more unique hands, which helps with keeping the XMult on Obelisk without risking playing the most played hand. Painted Deck also appreciates having such a high-value XMult Joker.
  • Blue Deck ( {{hl|blue|+1}} hand every round Discover at least {{hl|orange|20}} items from your [[collection]] Blue Deck.png)'s extra hands allow Obelisk to scale faster per round.
  • Checkered Deck ( Start run with {{hl|orange|26}} {{suit|Spades}} and {{hl|orange|26}} {{suit|Hearts}} in deck Win a run with the {{D|Black}} on any difficulty Checkered Deck.png) makes Flush easier to play, as the deck only has Spades and Hearts. Using this deck allows another option for the most played hand type other than Pair and Two Pair to build around and then pivot out of. It also helps the player after pivoting if the build does not revolve around Flush beforehand, as having another hand type that's easy to hit reduces the chance of resetting Obelisk accidentally.

Poker Hands

[edit | edit source]

No particular poker hand is better with Obelisk. The Pair or Two Pair strategy mentioned above could be considered a synergy, but the bulk of the game will be spent playing as many different hands as possible to keep options open in the future.

Jokers

[edit | edit source]
  • Jokers that produce Mult without hand-based conditions are what you always want to use with Obelisk. These can be flat + Mult Jokers, like Fibonacci (Each played {{hl|orange|Ace}}, {{hl|orange|2}}, {{hl|orange|3}}, {{hl|orange|5}}, or {{hl|orange|8}} gives {{Mult|+8}} when scored Uncommon Additive Mult On Scored Fibonacci.png) and Half Joker ({{Mult|+20}} if played hand contains {{hl|orange|3}} or fewer cards Common Additive Mult Independent Half Joker.png), but the most success comes from scaling ones, like Bootstraps ({{Mult|+2}} for every {{Money|5}} you have <br> {{Currently|{{Mult|+0}}}} Have at least {{hl|orange|2}} {{hl|edition|Polychrome}} Jokers at the same time. Uncommon Additive Mult Independent Bootstraps.png), Erosion ({{Mult|+4}} for each card below {{hl|orange|[the deck's starting size]}} in your full deck <br> {{Currently|{{Mult|+0}}}} Uncommon Additive Mult Independent Erosion.png), Ride the Bus (This Joker gains {{Mult|+1}} per {{hl|orange|consecutive}} hand played without a scoring {{hl|orange|face}} card <br> {{Currently|{{Mult|+0}}}} Common Additive Mult Mixed Ride the Bus.png), or Fortune Teller ({{mult|+1}} per [[Tarot Cards|{{hl|purple|Tarot}}]] card used this run <br> {{currently|{{hl|red|+0}}}} Common Additive Mult Independent Fortune Teller.png). Having one of these scaling Jokers will usually allow you to play all your hands per round, since you can place Obelisk before them for all but the last hand, and the last hand will have a large amount of extra Mult.
  • Spare Trousers (This Joker gains {{Mult|+2}} if played hand contains a [[Poker Hands|{{hl|orange|Two Pair}}]] <br> {{currently|{{mult|+0}}}} Uncommon Additive Mult Mixed Spare Trousers.png) will often naturally set up a stack of Two Pairs, making a smooth transition to other hands.
  • Jokers that provide scaling + Chips like Square Joker (This Joker gains {{Chips|+4}} if played hand has exactly {{hl|orange|4}} cards <br> {{Currently|{{Chips|0}}}} Common Chips Mixed Square_Joker.png) or Bull ({{Chips|+2}} for each {{$|1}} you have <br> {{currently|{{chips|+0}}}} Uncommon Chips Independent Bull.png), are very important to play Obelisk effectively, as few Chips will be provided from Planet Cards.
  • Vagabond (Create a [[Tarot Cards|{{hl|purple|Tarot}}]] card if hand is played with {{Money|4}} or less Rare Effect Independent Vagabond.png) and other Jokers that benefit from many hands being played per round work well with Obelisk, as you already want to play many hands per round to scale Obelisk. All of these kinds of Jokers in turn benefit from Acrobat ({{xmult|3}} on {{hl|orange|final hand}} of round Play {{hl|orange|200}} hands Uncommon Multiplicative Mult Independent Acrobat.png), as it provides a safety net for the last hand of the round.

Boss Blinds

[edit | edit source]
  • When a poker hand is not allowed to be played by The Psychic ( Boss The Psychic.png), The Eye ( Boss The Eye.png), or The Mouth ( Boss The Mouth.png), Obelisk will not reset or gain Mult. However, the hand counter in the Run Info will increase by 1. This theoretically allows a method of always increasing Obelisk.
  • Alternately, when Obelisk gets disabled by Crimson Heart ( Showdown Crimson Heart.png), the most played hands can be safely played without resetting Obelisk.

  • Nebula Deck ( Start run with the {{V|Telescope|class=hl-cyan}} voucher <br> {{hl|red|-1}} consumable slot Win a run with the {{D|Blue}} on [[Stakes|any difficulty]] Nebula Deck.png) has slight anti-synergy with Obelisk, as Telescope ([[Booster Packs|{{hl|orange|Celestial Packs}}]] always contain the [[Planet Cards|{{hl|cyan|Planet}}]] card for your most played [[Poker Hands|{{hl|orange|poker hand}}]] Base Telescope.png) encourages building around a single most-played hand type, which Obelisk discourages.

Jokers

[edit | edit source]
  • Burglar (When [[Blind|{{hl|orange|Blind}}]] is selected, gain {{hl|blue|+3}} Hands and {{hl|orange|lose all discards}} Uncommon Effect On Blind Select Burglar.png) removes all discards and may force the most played hand to be played, resetting Obelisk.
  • All Jokers that rely on playing specific hands, including Superposition (Create a [[Tarot Cards|{{hl|purple|Tarot}}]] card if poker hand contains an {{hl|orange|Ace}} and a {{ph|Straight}} <br> {{room}} Common Effect Independent Superposition.png), Crazy Joker ({{Mult|+12}} if played hand contains a {{ph|Straight}} Common Additive Mult Independent Crazy Joker.png), and The Order ({{xmult|3}} if played hand contains a {{ph|Straight}} Win a run without playing a {{ph|Straight}}.   • Planet Cards and Jokers that upgrade hands, like Space Joker ({{hl|green|1 in 4}} chance to upgrade level of played [[Poker Hands|{{hl|orange|poker hand}}]] Uncommon Effect On Played Space Joker.png) and Burnt Joker (Upgrade the level of the first {{hl|orange|discarded}} poker hand each round Sell 50 cards. Rare Effect On Discard Burnt Joker.png), are far less effective because the hand can only be used a limited number of times.

Vouchers
